Notes and scripts for Powershell Labs.
This Powershell lab is designed to help forensic investigation in Windows. We were introduced to Powershell 7. This lab identifies files that have been modified within the last 15 days and exports the results into a CSV file. 
This script also identifies suspicous files, reads a security event log and checks temporary files that were modified within
the last 600 hours. The goal is to detect suspcious activity and downloads that may need further review.
